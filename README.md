# Return-to-sports app

Python 2.7+
### Setup virtualenv
```
# Create virtualenv 
virtualenv env

# Activate virtualenv
source env/bin/activate

# Install all packages
pip install -r virtualenv_requirement.txt
```

### Development
```
# Activate virtualenv
source env/bin/activate

cd django

python manage.py runserver

127.0.0.1:8000/
```
